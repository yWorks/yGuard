yGuard

yGuard is an open-source Java obfuscation tool. With yGuard, it is easy as pie (üç∞) to configure obfuscation through an extensive ant task. This documentation explains how to use the yGuard Java obfuscation and shrinking software.

yGuard comes with a exhaustive set of examples to cover a broad range of use cases. When in doubt, look up the examples for solutions.
